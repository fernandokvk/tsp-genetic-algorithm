
# Algoritmo Genético Híbrido para o Problema do Caixeiro Viajante

Este trabalho consiste na pesquisa e implementação de um algoritmo baseado na meta-heurística de Algoritmos Genéticos (AG) combinado com mecanismos de busca local para o Problema do Caixeiro Viajante (PCV), ou, em inglês, Travelling Salesman Problem (TSP).

Algoritmos utilizados

 - Nearest neighbor search
 - 2-opt first improvement
 - Genetic Algorithm
     - Selection: Roulette, Ranking & Tournament
     - Crossover: OX1, PMX & POS
     - Update: Populational, Elitism & Steady stated
